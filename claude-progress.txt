- Start/pause/reset controls with keyboard shortcuts (Space, R, S)

- Pomodoro counter (X/4 format)

- Active task display - click any task to highlight it

1. No cloud sync - all data stored in localStorage only
2. Uses Web Audio beep (no actual audio files)
3. No drag-and-drop task reordering
4. Stats reset at midnight but no historical tracking beyond current day

## Technical Notes
- Pure vanilla JavaScript (no frameworks)
- CSS custom properties for theming
- IIFE module pattern for all JS modules
- LocalStorage keys: pomodoro_settings, pomodoro_stats, pomodoro_tasks, pomodoro_timer_state, pomodoro_active_task
